JKUAT Online Ip Telephony - Help

  1. To know the correct format to use for uploading campuses, departments and extensions kindly download a template for each entity.
  2. If you need a list of campuses, departments or extensions you can print either the pdf report or crystal reports.
  3. After creating a new user ensure that you have given them a role, else the system will not be able to authorize them.
  4. Incase you want to delete a user ensure that there is no role attached to them in users roles.
  5. Incase you want to delete a role ensure that no users are attached to that role and no rights are assigned to that role.
